# 버텍스 컬러 지정시
- 버텍스 페인트모드 -> 에디트(tab)
- 선택후 모드 복귀
- Shift + K 지정색으로 채움

# 버텍스 컬러를 뷰포트에 보이기
- 뷰포트 셰이드 - 솔리드
- 우측 아래방향 삼각형 - 컬러 - Vertex

# 솔리드모드에서 와이어 프레임 보이기
- object properties - viewport display - Show wireframe

# 단축키 (공용)
- A : 전체선택
- H : 숨김
- Alt + H : 숨김해제
- Alt + Z : 엑스레이모드
- Shift + Z : 와이어프레임

# 에디트모드 단축키
- M : 병합
- Shift + D : 선택한것 복제
- Alt + M : Weld
- V : 인접뜯어내기
- Y : 찢기
- P : 오브젝트 분리 (병합 : Ctrl+J)
- Alt + LMB : 루프 선택
- Shift + RMB : 커서
- Ctrl + L : 엘리먼트 선택
- Ctrl + F : tri to quad

# 백페이스 선택
- 와이어프레임 모드(Shift+Z)에서 선택 
- 아일랜드면 엘리먼트 선택
